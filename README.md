# HR ANALYTICS DASHBOARD


---

## ABSTRACT

Human Resource (HR) management plays a crucial role in the success of any organization. With the growth of employee data, traditional manual analysis becomes inefficient and error-prone. This project, **HR Analytics Dashboard**, focuses on transforming raw HR data into meaningful insights using data visualization techniques. The dashboard helps HR managers analyze employee performance, salary distribution, employment type, departmental trends, and cost metrics efficiently. By using an interactive dashboard, decision-making becomes faster, accurate, and data-driven.

## 1. INTRODUCTION

Human Resource departments handle large volumes of employee-related data such as age, salary, performance ratings, employment type, and department details. Analyzing this data manually is time-consuming and limits strategic planning. HR Analytics uses data visualization and analytical techniques to identify trends, patterns, and insights from HR data.

This project aims to design and develop an **HR Dashboard** that visually represents key HR metrics in an easy-to-understand format. The dashboard supports management in workforce planning, performance evaluation, and cost optimization.

---

## 2. OBJECTIVES OF THE PROJECT

* To analyze employee data using an interactive dashboard
* To visualize employee performance ratings and scores
* To identify salary cost distribution (minimum and maximum)
* To analyze employment type (Full-Time and Temporary)
* To support HR decision-making through data insights
* To reduce manual effort in HR reporting

---

## 3. SCOPE OF THE PROJECT

The scope of this project includes:

* Analysis of HR dataset
* Visualization of employee metrics
* Interactive filtering based on department, role, and employee type
* Dashboard creation for HR management use

The project does not include predictive analytics or real-time data integration.

---

## 4. DATASET DESCRIPTION

The dataset used in this project contains employee-related information collected from organizational HR records.

### Dataset Attributes:

* Employee ID
* Age
* Department
* Job Role
* Employment Type (Full-Time / Temporary)
* Salary / Cost
* Employee Rating (e-rating)
* Employee Score (e-score)
* Supervisor Score (s-score)

The dataset is stored in Excel format and used for visualization and analysis.

## 7. DASHBOARD DESCRIPTION

The HR Dashboard contains the following components:

### 7.1 Key Performance Indicators (KPIs)

* **Employee Rating (e-rating):** Overall employee rating value
* **Employee Score (e-score):** Performance-based score
* **Supervisor Score (s-score):** Supervisor evaluation score
* **Cost Minimum:** Minimum employee cost
* **Cost Maximum:** Maximum employee cost
* **Lowest Age:** Youngest employee age

### 7.2 Charts and Visualizations

* **Bar Chart:** Total salary or cost by job role (e.g., Area Sales Manager)
* **Pie Chart:** Department-wise or sales contribution
* **Line Chart:** Comparison between Production and Sales
* **Donut Chart:** Employment type distribution (Full-Time vs Temporary)

### 7.3 Filters (Slicers)

* Gender
* Employment Type
* Department
* Employee Status

These filters allow dynamic interaction with the dashboard.
